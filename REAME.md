# N7

A modern banking platform website built with React, Vite, and Tailwind CSS.

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```text
src/
├── assets/
├── components/
├── data/
├── layouts/
├── pages/
└── styles/
```

## Deployment

This project is configured to be deployed on Vercel.
